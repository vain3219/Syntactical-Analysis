## FIRST(α)
 * First(S)  = { type, i }
 * First(D)  = { type }
 * First(A)  = { i }
 * First(E)  = { (, i, num }
 * First(E') = { +, -, ε }
 * First(T)  = First(E)  = { (, i, num }
 * First(T') = { * , /, ε }
 * First(F)  = First(E)  = { (, i, num }


## FOLLOW(α)
 * Follow(S)  = { $ }
 * Follow(D)  = { $ }
 * Follow(A)  = { $ }
 * Follow(E)  = { ), $ }
 * Follow(E') = { ), $ }
 * Follow(T)  = { +, -, ), $ }
 * Follow(T') = { +, -, ), $ }
 * Follow(F)  = { * , /, +, -, ), $ }
